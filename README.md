## Завдання
Адаптувати програмний код статичного інтерфейсу до вимог фреймворка VueJS та забезпечити завантаження необхідних даних з Web-сервера. 

**Реалізований функціонал та технології:**
1. **Frontend:** Інтерфейс калькулятора (звичайний режим та режим для програміста) адаптовано під компоненти VueJS (через CDN). Реалізовано реактивність та двостороннє зв'язування даних.
2. **Клієнт-серверна взаємодія:** Асинхронні запити реалізовані за допомогою бібліотеки Axios.
3. **Backend:** Серверна частина написана на NodeJS з використанням фреймворка ExpressJS. Сервер обробляє реєстрацію, авторизацію та оновлення профілю користувача.
4. **База даних:** Дані користувачів зберігаються у локальній базі даних SQLite.

**Структура проєкту:**
* `index.html`, `login.html`, `register.html`, `profile.html`, `about.html` — клієнтські сторінки з Vue-компонентами.
* `server.js` — код серверної частини (NodeJS/ExpressJS).
* `database.sqlite` — файл бази даних SQLite (генерується автоматично).

## Інструкція із запуску
1. Встановіть необхідні залежності командою: `npm install express sqlite3 cors`
2. Запустіть сервер командою: `node server.js`
3. Відкрийте браузер за адресою: `http://localhost:3000/index.html`
